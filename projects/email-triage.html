AI Workflow

AI Email Triage Dashboard

An automated email classification system powered by Gemini AI. Reads incoming Gmail, classifies by category and urgency, suggests actions, and feeds a live auto-updating dashboard — so you stop sorting emails and start acting on them.

Google Apps Script Gmail API Gemini AI Google Sheets Dashboard

Why This Exists

The problem it solves

😩 Before

Business owners spending 1–2 hours daily reading, sorting, and prioritizing emails manually. Important requests buried under newsletters. Urgent issues missed. No visibility into email volume or patterns.

✨ After

Emails are automatically read, classified, and prioritized every 15 minutes. High-urgency items surface instantly. A live dashboard shows category distribution, urgency levels, and suggested actions at a glance.

Under The Hood

How it works

1

Fetch unread emails

Every 15 minutes, the script checks Gmail for new unread messages using time-driven triggers. Already-processed emails are skipped via Message ID tracking.

2

AI classification

Each email's subject and body are sent to Gemini AI for analysis. The AI returns a category (Sales, Support, Billing, Other), urgency level, summary, and suggested action. If the API is unavailable, a keyword-based fallback runs automatically.

3

Log to database

Results are batch-written to the Email Database sheet with timestamp, sender, classification data, and a direct Gmail thread link. Processed emails get labeled "Triaged" in Gmail.

4

Dashboard updates

The Dashboard sheet auto-updates using COUNTIF formulas — total volume, category breakdown, urgency distribution, and a filtered high-urgency table. Charts visualize the data in real time.
